#  Interactive Weather Dashboard

## Project Purpose
The Interactive Weather Dashboard is a modern weather forecasting application that
provides real-time weather information and a 5-day forecast for cities around the world. 
The project demonstrates the integration of external APIs, responsive UI design, and interactive frontend features.

The primary goal of this project is to practice frontend development concepts such as API integration, asynchronous JavaScript, responsive layouts, 
DOM manipulation, and user experience design while creating a practical real-world application.

## Features

###  City Search

* Search weather information for any city worldwide.
* Instant weather data retrieval using API requests.

### Current Weather Information

* Current temperature.
* Weather condition description.
* Dynamic weather icons.
* City location display.

###  Weather Highlights

* Wind Speed
* Humidity
* Visibility
* Atmospheric Pressure

### 5-Day Forecast

* Daily weather predictions.
* Temperature forecasts.
* Interactive forecast cards.

### Interactive Modal

* Click on forecast cards for detailed weather information.
* Smooth animations and transition

## Technologies Used

* HTML
* CSS
* JavaScript 
* OpenWeather API


## Installation


1. Download or clone this repository.
2. Open the project folder.
3. Enter a city name and start exploring weather data.


## API Integration

This project uses the OpenWeather API to retrieve:

* Current Weather Data
* 5-Day Weather Forecast
* Weather Icons
